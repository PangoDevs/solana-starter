# How to install Anchor

### What is Anchor?

Anchor is a Rust framework and will halp you to build smart contract.

### Installing Dependencies

To install and run Anchor you need install [Rust](./dependencies/rust), [Solana](./dependencies/solana), [NodeJS](./dependencies/node) and Yarn

#### Yarn

To install yarn run:

```shell
npm install -g yarn
```

#### Anchor

```shell
npm i -g @project-serum/anchor-cli
```

If you want build from source, check [this](https://project-serum.github.io/anchor/getting-started/installation.html#build-from-source-for-other-operating-systems) doc.

To check, you can initialize a new project running:

```shell
anchor init <name>
```
