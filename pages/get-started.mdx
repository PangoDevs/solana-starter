# Get Started

To work with Solana we need somes dependencies:

- [Solana](./dependencies/solana)
- [Rust](./dependencies/rust)
- [Anchor](./dependencies/anchor)
- Node
- Javascript
- TypeScript
- Postman

If you use Windows, follow this tutorial using a [WSL2](https://docs.microsoft.com/en-us/windows/wsl/install).
